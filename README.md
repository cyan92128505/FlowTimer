# 智慧番茄鐘應用
## FlowTimer - 不打擾的專注助手

---

## 專案概述

### 產品定位
一款基於 Flutter 開發的 macOS 桌面應用，提供智慧化的番茄鐘功能。透過監控用戶的實際操作頻率，動態調整專注時間，避免在工作流程中強制中斷用戶，同時融入輕量級遊戲化元素提升使用體驗。

### 核心理念
「尊重你的工作節奏，而非打斷它」

---

## 產品核心功能

### 1. 智慧時間調整
**傳統番茄鐘問題**：固定25分鐘，常在工作流程中強制中斷

**我們的解決方案**：
- 設定約25分鐘的彈性時間範圍
- 即時監控鍵盤滑鼠操作頻率
- 高頻操作時自動延長2-5分鐘（最多35分鐘）
- 零操作持續3分鐘以上提前結束並通知

### 2. 系統整合監控
- **輸入事件追蹤**：監控鍵盤滑鼠活動頻率
- **應用程式焦點**：識別當前工作應用
- **系統閒置檢測**：準確判斷用戶狀態
- **隱私保護**：僅統計頻率，不記錄具體內容

### 3. 遊戲化元素設計

#### 成長系統
- **虛擬植物培育**：每次專注為植物澆水，連續完成獲得陽光加成
- **等級進階**：從專注新手到禪定宗師，4個階段30+等級

#### 成就系統
- **趣味徽章**：夜貓子、早起鳥、心流騎士、鋼鐵意志等
- **隱藏成就**：摸魚大師、咖啡上癮等自嘲式成就
- **季節活動**：配合不同時期推出限定挑戰

#### 互動體驗
- **專注寵物**：Tamagochi 風格虛擬夥伴
- **專注地圖**：每日專注點亮生產力大陸
- **收集系統**：專注水晶、季節主題收集品

#### 數據視覺化
- **專注日曆**：GitHub 風格貢獻圖
- **年度回顧**：個人專注數據統計報告

## 技術實現

### 開發技術棧
- **前端框架**：Flutter 3.x（跨平台 UI）
- **原生整合**：Method Channel 橋接 macOS API
- **系統監控**：Core Graphics、IOKit、NSWorkspace
- **資料儲存**：SQLite 本地資料庫
- **後端服務**：Golang（可選，用於數據同步）

### 架構設計
```
Flutter UI Layer
       ↓
Method Channel Bridge
       ↓
Native macOS Module
       ↓
System APIs (Accessibility, IOKit)
```

### 開發時程
- **第1-3週**：MVP 開發（基本番茄鐘+活動監控）
- **第4-5週**：Beta 測試與反饋收集
- **第6-9週**：完整功能開發（智慧調整+遊戲化）
- **第10-11週**：上架準備與審核
- **第12週**：正式發布
